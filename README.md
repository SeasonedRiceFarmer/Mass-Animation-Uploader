# Mass Animation Uploader

Select any number of `KeyframeSequence`s in Explorer, pick a destination, and upload them all at once. Each successful upload creates an `Animation` object in your output folder with the correct `rbxassetid://` filled in. Source `KeyframeSequence`s are never modified.

[![YouTube Tutorial](https://img.youtube.com/vi/nTkhBOaF0Yw/maxresdefault.jpg)](https://www.youtube.com/watch?v=nTkhBOaF0Yw)

## Install

Download from the [Creator Store](https://create.roblox.com/store/asset/112038381420156) and install as a local plugin.

> Requires `CreateAssetAsync` beta enabled in Studio (`File → Beta Features`)

## Usage

1. Select `KeyframeSequence`s in Explorer
2. Pick a destination (personal account or group)
3. Pick an output folder
4. Click **Upload Selected**
